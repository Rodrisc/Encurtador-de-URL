# Encurtador-de-url

Um simples encurtador de links

# Instalações necessárias:
- Python 3
    #### No linux
    ``` 
    sudo apt install python3
    ```
    #### No windows
    Baixe o instalador no site oficial no Python:
    https://www.python.org/downloads/

- Flask
    ``` 
    pip install flask
    ```
